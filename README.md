
# Seu Remédio Vovó 🌸

Um assistente carinhoso e elegante para ajudar vovós a entenderem seus medicamentos e receitas médicas com a ajuda da Inteligência Artificial do Netinho.

## ✨ Como funciona?
- **Tire foto do remédio**: O netinho explica para que serve em linguagem simples e popular.
- **Leia receitas**: Identificamos os nomes dos remédios e buscamos farmácias REAIS próximas a você.
- **Localização Inteligente**: Encontramos as farmácias mais perto de você em Campo Grande ou qualquer lugar do Brasil.
- **Ouvir o Netinho**: A vovó pode ouvir a explicação em voz alta!

## 🚀 Como colocar na Internet (GitHub + Firebase)

Este projeto está pronto para o **Firebase App Hosting**, que é a forma mais moderna de publicar sites.

1. **GitHub**:
   - Crie um repositório novo no seu GitHub (ex: `meu-remedio-vovo`).
   - Envie este código para lá.

2. **Firebase Console**:
   - Vá em [console.firebase.google.com](https://console.firebase.google.com/).
   - Selecione seu projeto.
   - No menu lateral, clique em **App Hosting**.
   - Clique em **Começar** e conecte sua conta do GitHub.
   - Selecione o repositório que você criou.

3. **Pronto!**:
   - O Firebase vai criar um link (URL) para o seu site.
   - Toda vez que você mudar o código no GitHub, o site atualiza sozinho!

## 🛠️ Tecnologias
- **Next.js 15** (O motor do site)
- **Firebase Auth & Firestore** (Para guardar seus remédios com segurança)
- **Genkit AI** (A inteligência do netinho para ler fotos e textos)
- **Tailwind & ShadCN** (Para deixar tudo fofinho e elegante)
