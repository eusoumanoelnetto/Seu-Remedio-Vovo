
# Seu Remédio Vovó 🌸

Um assistente carinhoso e elegante para ajudar vovós a entenderem seus medicamentos e receitas médicas com a ajuda da Inteligência Artificial do Netinho.

## Como funciona?
- **Tire foto do remédio**: O netinho explica para que serve em linguagem simples.
- **Leia receitas**: Identificamos os nomes dos remédios e buscamos farmácias próximas.
- **Localização Inteligente**: Encontramos as farmácias mais perto de você em Campo Grande ou qualquer lugar do Brasil.

## Conectando ao GitHub e Firebase
Este projeto está pronto para o **Firebase App Hosting**. 

1. **GitHub**: Crie um repositório no seu GitHub e envie este código para lá.
2. **Firebase Console**: Vá em "App Hosting" no console do Firebase.
3. **Conectar**: Clique em "Começar" e selecione o seu repositório do GitHub.
4. **Pronto!**: O Firebase cuidará de publicar o site automaticamente sempre que o código no GitHub mudar.

## Tecnologias
- **Next.js 15** (O motor do site)
- **Firebase** (Onde guardamos seus dados com amor)
- **Genkit** (A inteligência do netinho)
- **Tailwind & ShadCN** (Para deixar tudo fofinho e elegante)
